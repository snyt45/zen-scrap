Zen Scrap で使えるキーボードショートカットと操作の一覧です。

---

## キーボードショートカット

| 操作 | ショートカット |
| --- | --- |
| ポスト・更新・保存 | `Cmd/Ctrl + Enter` |
| プレビュー切り替え | `Cmd/Ctrl + E` |

---

## ナビゲーション

| 操作 | 方法 |
| --- | --- |
| 一覧 / Inbox / コレクションの切り替え | ヘッダーのタブをクリック |
| スクラップを開く | 一覧でスクラップをクリック |
| コレクション詳細を開く | コレクション一覧でコレクションをクリック |
| アウトラインから移動 | アウトラインボタン → エントリをクリック（ハイライト付きでスクロール） |

---

## エントリ操作

| 操作 | 方法 |
| --- | --- |
| エントリの並べ替え | エントリ左側のドラッグハンドルをドラッグ&ドロップ |
| Inboxに追加/削除 | エントリヘッダーの受信箱アイコンをクリック |
| エントリのコピー | エントリメニュー → コピー |
| コレクションに追加 | エントリメニュー → コレクションに追加 |
| エントリの編集 | エントリメニュー → 編集 |
| エントリの削除 | エントリメニュー → 削除 |
| コードブロックのコピー | コードブロックにホバー → コピーボタン |

---

## スクラップ詳細

| 操作 | 方法 |
| --- | --- |
| タイトル編集 | タイトルをクリック → 入力して保存 |
| 説明の追加・編集 | 説明セクションの編集アイコン、または「+ 説明を追加」をクリック |
| タグ編集 | タグ行の編集アイコンをクリック（タグがない場合は「+ タグを追加」） |
| ステータス変更 | ステータスラベル（Open/Closed/Archived）をクリック → 選択 |
| ピン留め/解除 | アクションメニュー（…）→ ピン留め/ピン解除 |
| JSONコピー | アクションメニュー（…）→ JSONをコピー |
| ファイルを開く | アクションメニュー（…）→ ファイルを開く |
| コレクションに追加 | アクションメニュー（…）→ コレクションに追加 |
| スクラップの削除 | アクションメニュー（…）→ 削除 |

---

## コレクション

| 操作 | 方法 |
| --- | --- |
| コレクション作成 | コレクション一覧の「+ 新しいコレクション」ボタン |
| コレクション削除 | コレクション一覧のメニュー → 削除 |
| タイトル編集 | コレクション詳細でタイトルをクリック → 入力して保存 |
| アイテム追加（検索） | コレクション詳細の「+ 追加」→ スクラップを検索 → スクラップ全体またはエントリを選択（連続追加可） |
| アイテム追加（メニュー） | スクラップ/エントリのメニュー → コレクションに追加 → コレクションを選択 |
| アイテムの並べ替え | アイテム左側のドラッグハンドルをドラッグ&ドロップ |
| アイテムの削除 | アイテム右側の × ボタン |
| 元のスクラップに移動 | アイテムをクリック |
| まとめてコピー | 「まとめてコピー」ボタン（AI向けMarkdown形式でクリップボードにコピー） |

---

## Inbox

| 操作 | 方法 |
| --- | --- |
| Inboxに追加 | エントリヘッダーの受信箱アイコンをクリック |
| Inboxから削除 | 受信箱アイコンを再クリック、またはInbox一覧のチェックボックス |
| 消化（処理済み） | Inbox一覧でチェックボックスをクリック |
| 元のスクラップに移動 | Inbox一覧でエントリをクリック |

---

## 一覧画面

| 操作 | 方法 |
| --- | --- |
| 新規スクラップ作成 | 「+ 新規作成」ボタン |
| タグでフィルタリング | スクラップのタグをクリック |
| ピン留め/解除 | スクラップメニュー → ピン留め |
| アーカイブ/復元 | スクラップメニュー → アーカイブ |
| コレクションに追加 | スクラップメニュー → コレクションに追加 |

---

## 埋め込み

| 操作 | 方法 |
| --- | --- |
| 埋め込みの挿入 | アクションバーの「+ 埋め込み」→ 種類を選択 → URLを入力 |
| 対応形式 | X (Twitter)、YouTube、Web記事、GitHub |
| URL自動検出 | URLをペーストすると自動で埋め込み記法に変換（設定でオフ可） |
| 画像の添付 | アクションバーの「画像」ボタン、またはテキストエリアに画像をペースト |

| コマンド | 説明 |
| --- | --- |
| Open Zen Scrap | スクラップ一覧を開く |
