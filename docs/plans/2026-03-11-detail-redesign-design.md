# 詳細画面リデザイン設計

## 概要

Zennのスクラップ編集画面を参考に、詳細画面のヘッダー・空状態・エディタUI・埋め込み機能を改善する。

## 変更項目

### 1. ヘッダー改善

- 1行目: ステータスラベル（Open/Closed）+ 作成日 + エントリ数
- 2行目: タイトル（大きめ h2）+ 鉛筆アイコンボタン
- 鉛筆ボタンクリックでinline編集: タイトルがinputに変わり、保存/キャンセルボタン表示
- 「← 一覧」の戻るボタンは維持

### 2. 空状態表示

- エントリが0件のとき、タイムライン領域に「最初のコメントを追加しましょう」カードを表示
- 薄いボーダー、中央寄せ、テキストのみのシンプルなカード

### 3. エディタUI改善

- プレースホルダー: 「スクラップにコメントを追加」
- タブ: Markdown / Preview をpill風ボタンに変更（丸み、選択中は背景色付き、間に余白）
- 投稿ボタン: 「投稿する」、丸みのある青ボタン（右寄せ）

### 4. 埋め込み機能

- 「+ 埋め込み」ボタンをエディタ下部に配置
- クリックでドロップダウン表示（4種）:
  - X (Twitter): モーダルでURL入力 → `@[tweet](URL)` 挿入
  - YouTube: モーダルでURL入力 → URLからID抽出 → `@[youtube](ID)` 挿入
  - Web記事: モーダルでURL入力 → `@[card](URL)` 挿入
  - GitHub: モーダルでURL入力 → `@[github](URL)` 挿入
- モーダル: URL入力欄 + 挿入ボタン + キャンセルボタン
- 挿入後、textareaのカーソル位置に記法を追加
- Previewタブでzenn-markdown-htmlによりレンダリング確認可能

### 5. 画像アップロード

- 「画像」ボタンをエディタ下部に配置（埋め込みボタンの隣）
- クリックでファイル選択ダイアログ
- Obsidian Vault内に画像を保存 → `![](画像パス)` をtextareaに挿入
- Previewタブでプレビュー確認可能

## 対象外

- タグ機能
- コメント・返信機能
